#河北科技大学
# 2026Robocon-Volley-R1

本机器人只拥有远程遥控、自动打滑矫正、自动回球

底盘由VESC电调驱动5055电机形成等边三角形
上位机电流环:
Kp = 0.1
Ki = 200.0


双电机(灵足04电机)进行排球击球装置(作废)
力矩以及机构出现局限性，改为三气缸控制

遥控器使用ESP32-S3
遥控器的发送模块由HC12换为EBYTE的E32型(作废)
存在距离限制换为HC12

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

由于全向轮底盘在快速侧移时的物理现象：侧向加速时，轮子滚子与地面摩擦力不足，产生瞬时滑移 + 额外扭矩，导致机器人轻微偏航，而轮速命令仍是纯侧移，所以轨迹就“歪”了。缓慢推动时加速度小，摩擦力够用，就不明显。
引入JY61陀螺仪克服打滑现象

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

main.c--->freertos.c--->Task_Init.c开始任务调度
底盘运动解算:chassis.c内集成遥控器与力矩环
自动回球装置:hitball.c
遥控器(ESP32-S3)相关文件:(My_list)(comm)(dataFrame0(data_poll)(comm_stm32_hal_middle)
三次多项式(起初为了配合双电机击球装置(RobStride.c)使其达到最大速度):step.c
CAN总线通信:CANDrive.c
